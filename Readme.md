# Acrobits DemoPhone

This is a demo SIP phone application built using the Acrobits LibSoftphone SDK.
It showcases the capabilities of the SDK and provides a basic implementation of a SIP phone.

## Full Documentation
For full documentation, please visit the [Acrobits Softphone SDK Documentation](https://doc.acrobits.net/sdk/index.html).

## Getting Started
Please follow the quick start guide available at [Getting Started with Acrobits Softphone SDK](https://doc.acrobits.net/sdk/libsoftphone-android-ios/quick-start.html) to set up your development environment and run the demo application.
